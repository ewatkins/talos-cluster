# Media

Home media stack covering a media server, photo and document management, download automation, indexing, and notification tooling.

## Apps

| App | Description |
| --- | --- |
| [bazarr](bazarr/README.md) | Automatic subtitle downloader and manager for the media library |
| [immich](immich/README.md) | Self-hosted photo and video backup with face recognition and smart search |
| [jellyfin](jellyfin/README.md) | Media server for streaming movies, TV shows, and music with Intel GPU transcoding |
| [jellyseerr](jellyseerr/README.md) | Media request portal for Jellyfin, fulfilled automatically by Radarr and Sonarr |
| [notifiarr](notifiarr/README.md) | Notification aggregator for the *arr stack, forwarding events to Discord and other services |
| [paperless](paperless/README.md) | Document management system with OCR, OIDC authentication, and automated ingestion |
| [prowlarr](prowlarr/README.md) | Centralized indexer manager and proxy serving Sonarr and Radarr |
| [radarr](radarr/README.md) | Movie collection manager handling automated acquisition, sorting, and renaming |
| [recommendarr](recommendarr/README.md) | AI-powered recommendation engine suggesting new media based on existing library content |
| [sonarr](sonarr/README.md) | TV series collection manager handling automated acquisition, sorting, and renaming |
