# media

This namespace hosts the home media stack, including a media server, photo management, document management, indexing, download automation, and related tooling.

## Apps

- [bazarr](bazarr/README.md) - Subtitle downloading companion for Sonarr and Radarr
- [immich](immich/README.md) - Self-hosted photo and video backup and browsing
- [jellyfin](jellyfin/README.md) - Media server for streaming movies, shows, and music
- [jellyseerr](jellyseerr/README.md) - Media request management for Jellyfin
- [notifiarr](notifiarr/README.md) - Notification aggregator for the *arr stack
- [paperless](paperless/README.md) - Document management system with OCR
- [prowlarr](prowlarr/README.md) - Indexer manager and proxy for the *arr stack
- [radarr](radarr/README.md) - Movie collection manager and download automation
- [recommendarr](recommendarr/README.md) - AI-powered media recommendation engine for Sonarr and Radarr
- [sonarr](sonarr/README.md) - TV series collection manager and download automation
