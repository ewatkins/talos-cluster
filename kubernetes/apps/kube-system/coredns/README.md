# [CoreDNS](https://coredns.io/)

CoreDNS is the cluster DNS server, resolving service and pod names within the cluster. Its configuration is generated by talhelper and injected via a ConfigMap, ensuring it stays in sync with the Talos cluster configuration.

## Configuration

CoreDNS values are sourced from the `coredns-helm-values` ConfigMap generated by talhelper from `talos/talconfig.yaml`. To modify CoreDNS behavior (forwarders, custom zones, etc.), edit `talconfig.yaml` and regenerate with `talhelper genconfig`.
