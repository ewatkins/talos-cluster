# [Cilium](https://cilium.io/)

Cilium is an eBPF-based CNI plugin that provides high-performance networking, network policy enforcement, and deep observability via Hubble for Kubernetes clusters.

## Notes

- Helm chart: `cilium` v1.19.1 from the `cilium` Helm repository
- Hubble is enabled with relay and UI, exposed at `https://hubble.ewatkins.dev`
- Hubble metrics include: dns, drop, tcp, flow, port-distribution, icmp, http
- Additional Helm values are sourced from the `cilium-helm-values` ConfigMap (generated by talhelper)
- Prometheus ServiceMonitor enabled for both Cilium agent and Hubble relay
