# [Cilium](https://cilium.io/)

Cilium is the eBPF-based CNI for this cluster, providing pod networking, network policy enforcement, and deep network observability via Hubble. It is the foundation of all inter-pod and pod-to-service communication.

## Configuration

| Setting | Value | Notes |
| --- | --- | --- |
| Hubble UI | `https://hubble.ewatkins.dev` | Real-time network flow visualization across namespaces |
| Hubble metrics | dns, drop, tcp, flow, port-distribution, icmp, http | Scraped by Prometheus via ServiceMonitor |
| Additional values | `cilium-helm-values` ConfigMap | Generated by talhelper from `talos/talconfig.yaml`; includes KubeProxyReplacement and other low-level settings |

> Cilium's core networking configuration (KubeProxyReplacement, BPF settings, etc.) is managed in `talos/talconfig.yaml` and injected via the `cilium-helm-values` ConfigMap. Edit those settings there rather than directly in the HelmRelease.
